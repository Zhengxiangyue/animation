### Build

```bash
$ git clone https://github.com/Zhengxiangyue/animation.git
$ git checkout 1.0.1
$ cd animation/runtime
$ cmake ..
```

```bash
$ make
```

```bash
$ python ./motion_detection.py
$ ./playground
```
